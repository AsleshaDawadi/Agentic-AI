# Agentic AI Learning Process
This repository shows my ongoing project of learning agentic ai .


## Core Concepts I have Learned

### LLM Fundamentals
 The most basic thing I learned in this project was to get an API key of an LLM. Then I started learning and designing static and dynamic prompts. After learning the types of prompts I struggled through the main problem of AI — statelessness — and I started handling LLM state management to overcome stateless limitations. 

### Orchestration & Retrieval
After the fundamentals, I moved to the most highlighted part of agentic AI today — implementing basic LangChain workflows and sequential chains, and then building basic Retrieval-Augmented Generation (RAG) applications.
